Activity Type:     Existing Function

Activity Name:    Meals on Wheels

 

You are logged in as the Activity Manager who is responsible for completing a self-assessment of a activity identified above.  This assessment a number of sections as follows:

 

  • Purpose what is the purpose of the function/policy
  • Performance how well is the activity performing for each of the equality strands
  • Confidence what information is available to support the performance assessment
  • Additional Work what additional work is required to complete the impact assessment
  • Action Planning what are the necessary actions to address the issues identified by the assessment process

Each of the sections asks questions about the activity which you will be able answer by way of a drop down menu or a text box.  The (i) icon is the help facility. 

There are three pages to the tool.  This Home page.  The Overview page gives access to a progress tracking matrix.  The Summary page shows the overall level of completion of the tool for the activity.
